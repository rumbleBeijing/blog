# 博客使用说明
---
## 安装
```shell
//克隆代码到本地
git clone git@github.com:rumbleBeijing/blog.git

//切换到指定目录
cd blog

//安装依赖
yarn (npm install)
```
## 使用
本博客系统使用hexo构建，具体使用详见[hexo官网](hexo.io)。
```
//安装hexo
npm install hexo-cli -g

//创建一个md文件，
hexo new "你的文章题目"

//修改对应的文件

//发布
hexo deploy
发布的时候可能会需要登录github账号，bash可能不能弹窗github的登录窗口。 这一步建议使用windows的cmd工具完成。
```
